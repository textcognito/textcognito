# Textcognito

**Textcognito** is a modern, anonymous messaging platform that allows users to receive honest feedback and secret messages from friends safely. Users can create unique profile links to share on social media platforms like Instagram, Snapchat, and Twitter.

![Textcognito Preview](/public/og-image.jpg)

## üöÄ Features

* **100% Anonymous**: Complete privacy guaranteed. Sender identities are never revealed or stored.
* **Custom Links**: Users get personalized URLs (e.g., `textcognito.click/u/yourusername`) for easy sharing.
* **Smart Filters**: Advanced spam detection and content filters to keep inboxes safe (powered by AI).
* **Instant Delivery**: Real-time notifications for new messages.
* **Analytics Dashboard**: Track profile views and engagement statistics.
* **Responsive Design**: Built with a mobile-first approach using Tailwind CSS.
